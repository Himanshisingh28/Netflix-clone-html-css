##  Netflix Clone (HTML & CSS)

A responsive Netflix landing page clone built using pure HTML and CSS.  
This project replicates the UI design of Netflix homepage for learning and practice purposes.


##  Features

-  Netflix styled Hero Section
-  Movie Poster Row Section
-  Hover Effects on Movies
-  Navigation Bar with Sign In Button
-  Dark Theme UI
-  Basic Responsive Design

##  Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Hover Effects

## 📂 Project Structure

